# Katseye Official Website

## Project Overview
The Katseye website is a visually captivating and user-friendly platform designed to showcase the artist’s work and engage fans. It features sections dedicated to albums, merchandise, contact, and the artist’s background. The goal is to provide an interactive and seamless experience for fans, enabling them to explore, purchase, and connect effortlessly.

## Design Process
The website is tailored for **Katseye’s fans**, who seek an intuitive interface to access content such as albums, merchandise, and updates. 

### User Stories
- **As a fan**, I want to learn about Katseye's story, so that I can feel connected to the artist.
- **As a user**, I want to browse and purchase merchandise, so that I can support Katseye.
- **As a fan**, I want to stay updated on Katseye’s latest album, so that I can explore their music.
  
### Design Artifacts
Wireframes were created in Adobe XD, showcasing the layout for each page (Album, Merch, Contact, and About). These can be found in the accompanying `wireframes` directory.

## Features

### Existing Features
1. **Header Navigation**: Allows users to move between pages seamlessly with visually engaging hover effects.
2. **Album Page**: Displays the latest album with a bold title, description, and call-to-action button for purchase.
3. **Merch Page**: Showcases an interactive gallery of merchandise with clickable images redirecting to purchase links.
4. **Contact Page**: Features a functional contact form with JavaScript interaction for feedback upon submission.
5. **Footer**: Contains company info, social media links, and a "Back to Top" button with smooth scrolling functionality.

### Features Left to Implement
1. A fan forum or blog section for community interaction.
2. Integration of live updates for upcoming events and concerts.

## Technologies Used
- **HTML5**: For structuring the web content.
- **CSS3**: For styling, including responsive layouts and hover effects.
- **JavaScript**: To add interactivity, such as form submission feedback and smooth scrolling.
- **Google Fonts**: Used for a visually distinct typography style.
- **Adobe XD**: For designing and prototyping the website’s layout.

## Testing

### Scenarios Tested
#### Contact Form:
- Submitted an empty form → Displayed error messages for required fields.
- Submitted with an invalid email → Displayed error message for invalid input.
- Submitted valid inputs → Displayed "Thank you" message without reloading the page.

#### Cross-Browser Compatibility:
- Tested on Chrome, Firefox, and Edge for consistency.
- Ensured the website adapts to different screen sizes using responsive CSS.

#### Known Issues:
- A slight delay in the footer appearing on older browsers.
- Some hover effects are less smooth on mobile devices.
